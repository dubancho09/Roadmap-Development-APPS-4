# 🔥 Códigos HTTP Más Usados

## ✅ 2xx - Éxito

| Código | Significado    | Emoji   |
|--------|----------------|---------|
| 200    | OK             | ✅      |
| 201    | Created        | 🆕      |
| 204    | No Content     | 🚫📝    |

---

## 🟡 4xx - Error del Cliente

| Código | Significado        | Emoji   |
|--------|--------------------|---------|
| 400    | Bad Request        | ❌📝    |
| 401    | Unauthorized       | 🔒      |
| 403    | Forbidden          | ⛔      |
| 404    | Not Found          | 🔍❓    |
| 409    | Conflict           | ⚔️      |
| 422    | Unprocessable Entity | 🧩📭  |

---

## 🔴 5xx - Error del Servidor

| Código | Significado            | Emoji   |
|--------|------------------------|---------|
| 500    | Internal Server Error  | 💥🖥️    |
| 502    | Bad Gateway            | 🚪❌    |
| 503    | Service Unavailable    | 📴⚠️    |
| 504    | Gateway Timeout        | ⏱️🚪    |
